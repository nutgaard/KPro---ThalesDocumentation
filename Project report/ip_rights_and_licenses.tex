
\section{IP rights \& licenses}
We would like to use free-licenses that allows us to make a product where we donâ€™t have to give away our source code for free, but can sell our product to different customers. There are no problems in having to refer to the licenses we have used along the way, but we would like to decide the license for our product independent of the licenses we have used. 
\newline
\newline
Our customer has not said anything about what licenses we should use. They have however said that we must make it clear which parts of the code we have not done ourselves, and where we found the specific code.
